Backup & Restore Data

Info: Backup mencakup semua data: Produk, Transaksi, Pelanggan, Pengeluaran, dan Held Transactions.

Zona Berbahaya

Peringatan: Tindakan ini akan menghapus SEMUA data permanen. Tidak dapat dibatalkan.

Bantuan Sistem

Panduan Fitur

  • Terminal Kasir: Pilih produk, atur Diskon (Rp/%), toggle PPN 11%, input Nama & WA. Bayar tunai dengan kalkulasi kembalian otomatis. Struk tercetak & dikirim ke WA.
  • Tahan Transaksi: Tombol "Tahan" menyimpan keranjang sementara untuk melayani pelanggan lain. Klik kartu di bar kuning untuk resume.
  • Kategori Produk: Filter produk berdasarkan kategori. Search box untuk pencarian cepat.
  • PPN & Kembalian: Toggle PPN 11%. Untuk pembayaran tunai, input uang diterima → kembalian otomatis. Tombol quick cash 50K/100K/UANG PAS.
  • Pengeluaran: Catat biaya operasional. Sistem menghitung laba bersih = pendapatan - pengeluaran.
  • Produk Terlaris: Laporan menampilkan ranking produk berdasarkan jumlah terjual.
  • Data Pelanggan (CRM): Mencatat otomatis saat checkout. Broadcast promo WA ke semua pelanggan.
  • Export Laporan: Excel, PDF, Word, atau kirim ringkasan via WhatsApp.
